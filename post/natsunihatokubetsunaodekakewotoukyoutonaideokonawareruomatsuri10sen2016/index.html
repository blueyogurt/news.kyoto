夏には、特別なおでかけを。東京都内で行われるお祭り10選（2016）

2016.7.4 remove_red_eye711

夏のイベントとして外せないのがお祭り。屋台のグルメを食べるも楽しいし、浴衣を着て歩くだけでもテンションがupしますよね♪そんな夏の風物詩ともいえる東京都内のお祭りをまとめて紹介します♡

2016年の東京のお祭りはチェックした？

via kura3.photozou.jp
夏の定番イベントと言えば、お祭り。小さいころは近くの神社や公園に行って楽しんだりした思いではありませんか？大人になった今、行ってみるのも楽しいもの。浴衣を着て彼とデートしたり、お友達と屋台グルメを楽しんでみたり…♡今回は、東京都内で行われるお祭り紹介します！

千代田区納涼の夕べ

via livedoor.blogimg.jp
最後はちょっとしんみりとしたとうろう流しのお祭りを。千代田区納涼の夕べでは皇居の森や北の丸公園を背景として約700コのとうろうがお濠にゆらめきます♪

via car-moby.jp
ちなみにこのとうろうを浮かべたい方は事前予約が必要。ボート場から出るボートに乗って浮かべると、ラプンツェルの世界のような気分が味わえますよ！
＜千代田区納涼の夕べ＞
■開催日：
2016年7月15日
■開催場所：
千鳥ケ淵緑道内 千鳥ケ淵ボート場
詳細はこちら

浅草夏の夜まつり とうろう流し

via www.wakasaji-navi.com
都内のとうろう流しの代名詞と言えば隅田川。毎年恒例のとうろう流しは今年ももちろん開催♪吾妻橋近くの隅田公園親水テラスから流すと、スカイツリーのライトアップも相まって幻想的な雰囲気に♡

via e-asakusa.jp
ちなみにとうろうは専用のスロープを使って流すため、キレイに流れていくのも特徴。フォトジェニックなシーンが摂れること間違いなしです！
＜浅草夏の夜まつり とうろう流し＞
■開催日：
2016年8月13日
■開催場所：
台東区隅田公園親水テラス
詳細はこちら

うえの夏祭り

via trend-trend.com
上野の夏の風物詩ともいえるお祭り。週末になるとパレードやサルのショーなど、様々なイベントで盛り上げてくれるので何度でも行きたくなっちゃいますよ！うえの夏祭りと言えば、数多くの屋台が並ぶことでも知られています。期間も長いため、お仕事帰りにふらりと立ち寄ってみるのもいいかも♡
＜うえの夏祭り＞
■開催日：
2016年7月9日～8月7日
■開催場所：
上野公園、不忍池など
詳細はこちら

乃木神社の夏祭り

via untomiryoku.xsrv.jp
昔懐かしい神社のお祭りを体験するなら乃木坂神社がイチオシ。境内には輪投げやヨーヨーなどの昔懐かしい屋台が並んでいるので、思いっきり童心に帰れちゃいますよ♪さらに照明や衣装などの演出に請っているのも乃木神社ならではのスタイル。浴衣の着付け教室もあるので、ここで浴衣デビューしちゃうのもいいかも!?
＜乃木神社の夏祭り＞
■開催日：
2016年8月6日
■開催場所：
乃木神社境内
詳細はこちら

麻布十番納涼まつり

via blogs.c.yimg.jp
下町風情が残りつつ、国際色豊かな街として知られる麻布十番。毎年恒例となった納涼まつりが今年も開催されます。写真で見てもわかるように大人気イベントとなっています！麻布十番納涼まつりでは12年から出店を商店街のお店に限定しているのもポイント。屋台の味が気に入ったら、今度は麻布十番で飲みに行く…なんて出会いもできちゃいますよ♪
＜麻布十番納涼まつり＞
■開催日：
2016年8月27日～8月28日
■開催場所：
麻布十番商店街
詳細はこちら

ほおずき市

via www.tenki.jp
東京の夏を彩る植物と言えばほおずき。観音信仰と関連の深いアイテムとされていて、中でも7月10日に行ってほうずきを購入すると4万6千日分の参拝効果があるのだとか！何かお願い事がある人はぜひとも行ってみたいところですね♪

ちなみにほおずきの鉢植えを買ってしばらく育てるとこうした形になるのだとか。繊維のきめ細やかさが見られて、1度で2度楽しめちゃいますよ♪
＜ほおずき市＞
■開催日：
2016年7月9日～7月10日
■開催場所：
浅草寺
詳細はこちら

神楽坂まつり

via darkeye.biz
料亭などが立ち並び、路地裏をただ歩くだけでも楽しい神楽坂の夏のイベント。浅草寺でほおずきが買えなかった場合はここでも購入可能。夏の風物詩をゲットするには最適ですね♪ちなみに神楽坂まつり最大のハイライトとなるのが阿波踊り。出場チームの一糸乱れぬ踊りを堪能できちゃいますよ♡
＜神楽坂まつり＞
■開催日：
2016年7月27日～7月30日
■開催場所：
神楽坂1丁目～神楽坂通り
詳細はこちら

神田明神七夕祭り

via hibinote.net
夏らしいイベントと言えば、欠かせないのが七夕。短冊にお願い事を書いて笹に飾った思い出は誰もがやったことがあると思いますが、それを大人になった今でもやるならば神田明神がオススメ。縁結びの神様がいるパワースポットだけに効果も絶大ですよ♡

via sugoroku.tokyo
さらに神田明神では七夕限定の縁結びお守りを授与してくれるサービスも。キュートなお守りをもらってお願い事を叶えちゃいましょう♪
＜神田明神七夕祭＞
■開催日：
2016年7月7日
■開催場所：
神田明神
詳細はこちら

福生七夕祭り

via pds.exblog.jp
今年の七夕当日は木曜日。仕事が忙しくて行けないという方にはこちらの七夕祭りがオススメ。通りには100店以上の出店が並び、ただ歩くだけでも楽しめちゃいます♪市民のお祭りと言うニュアンスも強い福生七夕祭りでは織姫彦星コンテストなどの市民参加型のイベントもたくさん。催しに参加してみるのも楽しいですよ♡
＜福生七夕まつり＞

■開催場所：

阿佐谷七夕まつり

via blogimg.goo.ne.jp
日本三大七夕祭りとして知られる阿佐ヶ谷の恒例イベント。駅周辺の9つの商店街には短冊がたくさんつけられた笹と名物のハリボテ飾りが並んでいます♪ちなみにハリボテ飾りは1つだけでないのがポイント。思い思いのハリボテ飾りを写真で撮るだけでも楽しいですよ！
＜阿佐谷七夕まつり＞
